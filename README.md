# COVID19
코로나현황판 

## 기능 상세
- 시도별 신규 확진자 수가 파이차트로 표시
- 도시 항목을 선택하면 상세 현황을 볼 수 있는 화면으로 이동


## 활용기술 
- 굿바이 코로나 19 API
- Alamofire
- Cocoapods
